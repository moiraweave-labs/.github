# MoiraWeave Labs Organization

Shared GitHub metadata and governance for the MoiraWeave organization.

## Contents

This repository contains:

- **Profile**: Organization profile and README displayed on the organization home page
- **Templates**: Issue and pull request templates for standardized workflows
- **Policies**: Code of conduct, security policy, and contribution guidelines
- **Governance**: Organization-wide configuration and standards
